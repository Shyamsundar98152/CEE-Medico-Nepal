JSON Format

Your JSON must follow one of these two formats:

Format A — Plain array:

[
  {
    "question": "What is...?",
    "options": ["A", "B", "C", "D"],
    "correct": 2,
    "explanation": "Because..."
  }
]

Format B — Object with metadata:

{
  "name": "Zoology Set 1",
  "subject": "Zoology",
  "questions": [ /* same as Format A */ ]
}

correct is a 0-based index of the correct option. explanation is optional.

Firebase Rules

Set these rules in your Firebase console (Realtime Database → Rules):

{
  "rules": {
    "mcq_sets": {
      ".read": true,
      ".write": false
    }
  }
}

Public read, no public write. Uploads go through this admin page only.
