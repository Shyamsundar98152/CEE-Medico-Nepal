JSON Format

Format A — Plain array:

[
  {
    "question": "Which nerve supplies...?",
    "options": ["A","B","C","D"],
    "correct": 2,
    "explanation": "Because..."
  }
]

Format B — With metadata:

{
  "name": "Anatomy Set 1",
  "subject": "Anatomy",
  "questions": [/* array above */]
}
Firebase Rules
{
  "rules": {
    "mcq_sets": {
      ".read": true,
      ".write": true
    }
  }
}

Set .write: false in production for security.
